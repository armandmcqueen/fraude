# Fraude

A simple chat application built on Claude.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Set your Anthropic API key:
   ```bash
   export ANTHROPIC_API_KEY=your-key-here
   ```

## Development

Start the dev server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Testing

Run all tests:
```bash
npm run test
```

Run live LLM tests (requires API key, hits real API):
```bash
npm run test:live
```

## Linting

```bash
npm run lint
```
